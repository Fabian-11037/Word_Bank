Importación de Datos

A continuación, realizamos una descripción breve de los campos de información encontrados en cada una de los datasets trabajados:

temperature_data <- read_csv("./Data/data.csv", skip = 3)
emissions_data <- read_csv("./Data/annual-co2-emissions.csv")
emissions_per_capita_data <- read_csv("./Data/co2-emissions-per-capita.csv")
climate_change_data <- read_excel("./Data/climate_change.xlsx", sheet = "Data")
country_data <- read_excel("./Data/climate_change.xlsx", sheet = "Country")
series_data <- read_excel("./Data/climate_change.xlsx", sheet = "Series")

Dataset ‘Annual CO2 Emissions’

Obtenido de (Ritchie et al., 2023), este dataset nos provee información histórica de las emisiones totales de cada entidad o país desde el año 1949.

Variable Descripción
Entity País o agrupación de paises
Code Código ISO-Alpha 3 (país) u OWID (agrupación) de la entidad
Year Año de muestreo del dato
CO2 emissions Total emitido por la entidad en el año en cuestión en Toneladas

Dataset ‘CO2 Emissions per Capita’

Obtenido de (Ritchie et al., 2023), este dataset nos provee información histórica de las emisiones per capita de cada entidad o país desde el año 1949.

Variable Descripción
Entity País o agrupación de paises
Code Código ISO-Alpha 3 (país) u OWID (agrupación) de la entidad
Year Year of the data
CO2 emissions per capita Total emitido por la entidad dividido por la población en el año en cuestión en Toneladas por persona

Dataset ‘Global Land and Ocean 12-Month Period Average Temperature Anomalies’

Obtenido de (NCEI, 2026), este dataset nos provee información histórica de la temperatura anómala global promedio por mes desde diciembre de 1850.

Variable Descripción
Date Mes de muestreo del dato en formato YYYYMM
Anomaly Temperatura anómala, calculada como la diferencia entre la temperatura global promedio y la temperatura global promedio entre los años 1861 a 1890, en Celsius

Dataset ‘Climate Change Data’

Obtenido de (World Bank, 2023), este dataset nos provee información histórica diferentes variables climaticas y geopolíticas de los países del mundo, desde el año 1990.

Variable Descripción
Country code Código ISO-Alpha 3 (país) u OWID (agrupación) de la entidad
Country Name Nombre de la agrupación o país
Series Code Código de la variable muestreada
Series Name Nombre de la variable muestreada
1990 - 2011 Columnas que corresponden al año de muestreo. Para algunas variables, como proyecciones a futuro, solo se encuentra data en la columna 2011, correspondiente al valor de la proyección
